1. Autenticación y Gestión de Usuarios

1.1 Autenticación
POST /api/auth/registro/ - Registro de usuarios normales
POST /api/auth/iniciar-sesion/ - Inicio de sesión de usuarios
POST /api/auth/token/refrescar/ - Refrescar token JWT
GET /api/auth/verificar-rol/ - Verificar rol del usuario actual

1.2 Empleados
POST /api/auth/empleado/registro/ - Registro de empleados (solo admin)
POST /api/auth/empleado/iniciar-sesion/ - Inicio de sesión empleados

1.3 Administradores
POST /api/auth/admin/iniciar-sesion/ - Inicio de sesión administradores

1.4 Perfil
GET/PUT /api/perfil/ - Ver/actualizar perfil del usuario

2.1 Usuarios (Admin)
GET /api/admin/usuarios/ - Listar usuarios
POST /api/admin/usuarios/ - Crear usuario
GET /api/admin/usuarios/{id}/ - Ver detalle usuario
PUT /api/admin/usuarios/{id}/ - Actualizar usuario
DELETE /api/admin/usuarios/{id}/ - Eliminar usuario
POST /api/admin/usuarios/{id}/activar/ - Activar usuario
POST /api/admin/usuarios/{id}/desactivar/ - Desactivar usuario

2.2 Sucursales (Admin)
GET /api/admin/sucursales/ - Listar sucursales
POST /api/admin/sucursales/ - Crear sucursal
GET /api/admin/sucursales/{id}/ - Ver detalle sucursal
PUT /api/admin/sucursales/{id}/ - Actualizar sucursal
DELETE /api/admin/sucursales/{id}/ - Eliminar sucursal

2.3 Servicios (Admin)
GET /api/admin/servicios/ - Listar servicios
POST /api/admin/servicios/ - Crear servicio
GET /api/admin/servicios/{id}/ - Ver detalle servicio
PUT /api/admin/servicios/{id}/ - Actualizar servicio
DELETE /api/admin/servicios/{id}/ - Eliminar servicio

2.4 Configuración (Admin)
GET /api/admin/configuracion/ - Listar configuraciones
POST /api/admin/configuracion/ - Crear configuración
GET /api/admin/configuracion/{id}/ - Ver detalle configuración
PUT /api/admin/configuracion/{id}/ - Actualizar configuración
DELETE /api/admin/configuracion/{id}/ - Eliminar configuración

3. Gestión de Turnos
3.1 Turnos (Usuario)
POST /api/turnos/ - Crear turno
GET /api/mis-turnos/ - Listar turnos del usuario
GET /api/mis-turnos/{id}/ - Ver detalle de turno
GET /api/turnos/agenda/ - Listar turnos agendados
GET /api/turnos/historial/ - Ver historial de turnos

3.2 Gestión de Turnos (Empleado)
POST /api/empleado/turnos/siguiente/ - Llamar siguiente turno
POST /api/empleado/turnos/{id}/completar/ - Completar turno
POST /api/empleado/turnos/{id}/transferir/ - Transferir turno
GET /api/empleado/turnos/actual/ - Ver turno actual
GET /api/empleado/colas/ - Ver cola de turnos
GET /api/empleado/estadisticas/ - Ver estadísticas

3.3 Calificaciones
POST /api/calificaciones/crear/ - Crear calificación
GET /api/calificaciones/ - Listar calificaciones
GET /api/calificaciones/{id}/ - Ver detalle calificación

4. Reportes y Listados Públicos
GET /api/admin/reportes/ - Generar reportes avanzados
GET /api/servicios/ - Listar servicios disponibles
GET /api/sucursales/ - Listar sucursales activas

5. WebSocket
WS /ws/turnos/ - Websocket para actualizaciones en tiempo real
Para probar estos endpoints en Postman, recuerda:

Para endpoints protegidos, incluir el header:
Authorization: Bearer <token_jwt>

Para las peticiones POST/PUT, usar el Content-Type:
Content-Type: application/json

El token JWT se obtiene al iniciar sesión y debe ser renovado usando el endpoint de refrescar token cuando expire.
